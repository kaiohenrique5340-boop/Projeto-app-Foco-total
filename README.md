# Projeto-app-Foco-total
 Foco Total 📱🧠

Foco Total é um aplicativo mobile desenvolvido com React Native + Expo que ajuda usuários a reduzir distrações digitais, melhorar o foco e desenvolver hábitos mais saudáveis de uso do smartphone.

O app monitora o uso de aplicativos, permite definir metas de bem-estar digital e utiliza inteligência artificial para gerar intervenções personalizadas que ajudam o usuário a manter o foco.

🚀 Objetivo

O objetivo do Foco Total é ajudar pessoas que sofrem com uso excessivo do celular, oferecendo ferramentas práticas para:

Monitorar tempo de uso de aplicativos

Criar metas de bem-estar digital

Receber insights inteligentes sobre hábitos digitais

Bloquear apps distrativos durante períodos de foco

Receber intervenções motivacionais baseadas em IA
